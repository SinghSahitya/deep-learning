Robust Deepfake Video Detection under Adversarial Conditions
Problem Statement: Deepfake videos pose serious threats by enabling hyper-realistic misinformation. Yet current detectors struggle when attackers introduce subtle perturbations to evade detection [1,2]. This project tackles the gap by proposing an adversarial robust deepfake detector. We seek to integrate adversarial defense techniques into deepfake classification, so the model remains accurate even when fake videos are intentionally modified to fool it. The core idea is to learn invariant features that distinguish real vs. fake faces despite compression artifacts and adversarial noise [3,4]. By addressing adversarial attacks (e.g., imperceptible pixel perturbations) [5,6], the project aims to enhance the reliability of deepfake detection in real-world, hostile conditions.
Key Papers:
Khan et al. (2024), “Adversarially Robust Deepfake Detection via Adversarial Feature Similarity Learning,” arXiv 2403.08806. (Introduces a loss function to train deepfake detectors robust against adversarial perturbations)[1][2].
Yan et al. (2024), “Transcending Forgery Specificity with Latent Space Augmentation for Generalizable Deepfake Detection,” CVPR 2024. (Augments latent features to improve detection generalization across different fake-generation methods)[7][8].
Exposing DeepFakes via Hyperspectral Domain Mapping (arXiv 2511.11732): spectral domain deepfake detection showing promise over RGB-only baselines
Expected Research Outcome: A deepfake video detection model that achieves high accuracy on standard deepfake datasets and maintains performance against adversarial attacks (e.g., FGSM, PGD). The research component includes a new loss or training scheme that jointly optimizes detection and adversarial robustness (e.g., feature similarity regularization or adversarial training)[9][10]. The team will evaluate robustness using established benchmarks and Auto Attack. The demo will be an interactive Colab notebook where users upload a video and the system outputs a robustness-aware real/fake verdict, illustrating stable detection even when videos are adversarially perturbed.
