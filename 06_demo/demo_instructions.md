# Demo Instructions

To run the interactive Deepfake Detection web interface:

### 1. Requirements
Ensure you have activated your python environment and installed the dependencies.
```bash
pip install -r 03_code/requirements.txt
```

### 2. Trained Model Weights
The demo actively assumes the existence of the `best_efficientnet.pth` checkpoint spanning all validation processes.
- **Location required**: `05_results/models/best_efficientnet.pth`
- If you haven't trained a model yet, you must complete `python 03_code/scripts/train.py` first. Or run the demo and it will attempt to initialize with random weights.

### 3. Launch the Demo
Execute the UI script:
```bash
python 03_code/scripts/demo.py
```

### 4. Accessing the Web UI
Since the script uses Gradio with `share=True`, it will print two URLs to your terminal:
- A local URL (e.g., `http://127.0.0.1:7860`)
- A public URL (e.g., `https://xxxxx.gradio.live`)

Go to the **public URL** generated by Gradio in your computer's browser. You will be able to upload a video, and the EC2 Server GPU will process it and display the outcome in your browser!
